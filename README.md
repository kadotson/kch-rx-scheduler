# KCH Rx Scheduler

Internal scheduling prototype as a phone-installable web app (PWA).

## Current version
This version runs as a React web app and uses in-app alerts only. Firebase config is included, but the current UI is still using local in-browser state. The next development step is wiring Firestore collections so data saves and syncs across employees.

## How to run locally
1. Install Node.js
2. Run `npm install`
3. Run `npm run dev`

## How to deploy
Use GitHub Pages or Firebase Hosting after the app builds successfully.
